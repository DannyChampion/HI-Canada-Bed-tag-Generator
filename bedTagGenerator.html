How to Use

  1. Go to GuestCentrix Reports and select the Arrival List by Date Range report.
  2. Set the date to today and sort by Room No.
  3. Export the report as a .csv file.
  4. Click “Browse” above and select the exported file.
  5. Review the generated bed tags below for discrepancies
  6. Click "Print tags" and set paper size to A6
