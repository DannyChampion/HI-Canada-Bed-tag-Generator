Guest Bed Tags Generator

How to Use

  1. Go to GuestCentrix Reports and select the Arrival List by Date Range report.
  2. Set the date to today and sort by Bed No..
  3. Export the report as a .csv file.
  4. Click “Upload Guest CSV File” above and select the exported file.
  5. Review the generated bed tags below.
  6. Press ctrl+p to print and set paper size to A6
